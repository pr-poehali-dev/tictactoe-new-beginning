# Крестики-нолики: Игровая спецификация

## 1. Монетизация: Общая схема

| Инструмент | Описание | Триггер покупки |
|---|---|---|
| Монеты | Внутренняя валюта. Покупаются за реальные деньги пакетами (100 / 500 / 1500). Начисляются за победы в PvP и ежедневный вход. | Ускорение прогресса, ставки, скины. |
| Подписка PRO | Ежемесячная. Отключает рекламу, открывает детальную статистику, 3D-режим, «Режим Анализа» матча. | Комфорт, эксклюзивный контент, хардкор. |
| Магазин скинов | Платные облики для X/O и поля. | Самовыражение. |
| Бустер-паки карт | Лутбоксы со случайными картами способностей. | Азарт, тактическое преимущество. |
| Боевой пропуск | Сезонный (месяц). Содержит уникальные скины и карты, открывается за грин или покупку уровней. | Удержание, эксклюзивность. |
| Комиссия платформы | 10% с дуэлей на монеты, ставок зрителей на турнирах, продаж UGC-скинов. | Пассивный доход. |

## 2. Регистрация и Личный кабинет

### Регистрация
- Поля: Логин, Email, Пароль.
- OAuth: Google, VK, Telegram.
- Стартовый бонус: 50 монет + базовый бустер-пак (3 обычные карты).

### Личный кабинет (Dashboard)
- **Профиль:** Аватар, Ник, Ранг, Уровень, Elo-рейтинг.
- **Ресурсы:** Счётчик Монет, Очков Славы, Билетов Безумия, Энергии PvE.
- **Статистика:** Общий винрейт, график рейтинга, тепловая карта ходов (PRO), история матчей.
- **Инвентарь:** Коллекция скинов, Колода карт (просмотр и сборка), Собранные фрагменты UGC.
- **Быстрые действия:** «В бой», «Открыть бустер», «Колесо фортуны».

## 3. Игровые режимы

### А. Сражения (PvP)
- **Быстрый бой.** Подбор по Elo. Поле 3x3. Таймер 15 сек/ход. Награда: +Рейтинг, +10 монет за победу.
- **Дуэль на интерес.** Игроки вносят в банк от 10 до 500 монет. Победитель забирает банк за вычетом 10% комиссии.
- **Арена 4x4 / 5x5.** Сбор 3-х (4-х) в ряд. Доступно всем. Повышенный риск, выше награда.

### Б. Безумие (Twist Rules)
Вход: 1 Билет Безумия. Ротация правил раз в 3 дня.
- **Гравитация:** знаки падают вниз.
- **Шахматный блиц:** 30 сек на всю партию.
- **Порча:** возможность удалить знак соперника раз в 2 хода.
- **Мутатор:** поле случайной формы с заблокированными клетками.

### В. Сюжет (PvE)
- Карта мира. Линейные главы с уникальными боссами (боты с читерскими способностями).
- Энергия: 1 игра = 1 ед. энергии. Мах запас 10. Восстановление: 1 ед. / 30 мин.
- Награда: Очки Славы, эксклюзивные PvE-скины.

### Г. Турниры (События)
- **Субботняя королевская битва.** Сетка на 64/128 игроков, игра на выбывание. Поле 4x4.
- **Ставки зрителей.** Не участвующие игроки могут ставить монеты на фаворитов, получая выигрыш с комиссией платформы.

## 4. Система карт способностей (ККИ-элемент)

- **Колода:** 5 карт перед матчем. Карты одноразовые в рамках одного боя.
- **Энергия маны:** На старте 0, +1 каждый ход.

### Карты
| Карта | Стоимость | Эффект |
|---|---|---|
| Блок | 1 мана | Запретить сопернику ход на выбранную клетку на 1 ход |
| Сдвиг | 2 маны | Сместить ряд на 1 позицию |
| Заморозка | 3 маны | Таймер хода соперника сокращается вдвое |
| Джокер | 4 маны | Поставить универсальный знак |
| Ультиматум | 5 маны | Стереть один любой знак соперника с поля |

- **Редкость:** Обычная, Редкая, Эпическая, Легендарная (влияет на анимацию, не на силу).
- **Источник:** Покупка бустер-паков, награды Боевого пропуска, Колесо фортуны.

## 5. UGC и Кастомизация

**Конструктор скинов** — инструмент пиксель-арта 16x16 / 32x32 внутри личного кабинета.

**Процесс:**
1. Игрок рисует «X» и «O».
2. Отправляет на премодерацию (ИИ-фильтр + ручная проверка).
3. Одобренный скин попадает в Мастерскую.
4. Другие игроки покупают его за Монеты.
5. Автор получает 5% роялти, платформа — 95%.

---

## 6. Боевой пропуск и Колесо фортуны

### Боевой пропуск (Сезонный, 30 дней)
- **Бесплатная ветка:** Обычные карты, 100 монет.
- **Премиум ветка** (покупка за донат): Легендарные скины, редкие карты, Билеты Безумия.
- **Прокачка:** За выполнение заданий (сыграть 3 игры в Безумии, победить 5 раз за X, и т.д.).

### Колесо фортуны
- Стоимость: 10 монет / 1 бесплатный прокрут в день.
- Награды: Случайные монеты, бустеры, фрагменты редких скинов, Билеты.

## 7. Технический стек и архитектура

- **Backend:** Node.js (NestJS) или Python (FastAPI). Логика ботов, рейтинг, магазин, матчмейкинг — REST API.
- **Real-time:** Socket.IO для PvP, турниров, чата.
- **Database:** PostgreSQL (профили, инвентарь, транзакции). Redis (сессии, очереди матчмейкинга, кэш рейтинга, временные игровые комнаты).
- **Frontend:** React / Vue 3. Canvas (PixiJS) для рендера поля, анимаций скинов и карт. CSS Grid для статики.
- **DevOps:** Docker, Nginx, CI/CD на GitHub Actions.
- **Платежи:** Stripe / ЮKassa.

---

## 8. Защита от читов и мультиаккаунтинга

- **Валидация ходов:** Вся логика игры на сервере. Клиент присылает только номер клетки. Сервер сам проверяет очерёдность, выход за пределы и дубликат хода.
- **Антифрод:** Лимит на вывод монет из «бесплатного» пула аккаунтов в дуэлях. Анализ цепочек переводов между аккаунтами.
- **Капча:** При подозрительной частоте создания аккаунтов.

## 9. Схема API (ключевые эндпоинты)

### Аутентификация
```
POST /api/auth/register
POST /api/auth/login → JWT Token
```

### Пользователь
```
GET /api/user/profile → баланс, рейтинг, инвентарь
GET /api/user/stats   → детальная статистика (PRO)
```

### Магазин и инвентарь
```
GET  /api/store/skins          → список скинов с ценами
POST /api/store/buy/skin
GET  /api/store/lootboxes
POST /api/store/open/lootbox
```

### Игровой процесс (WebSocket события)
```
client → server: "search_game"  { mode: "ranked" | "duel" | "madness" }
server → client: "match_found"  { opponent, room_id }
client → server: "make_move"    { room_id, cell, card_id (optional) }
server → client: "game_state"   { board, turn, mana }
server → client: "game_over"    { result, reward }
```

### UGC
```
POST /api/ugc/upload  → отправка пиксель-арта на модерацию
GET  /api/ugc/gallery → витрина одобренных скинов
```
